# My dotfiles
## Install
```shell
$ curl -L git.io/watc_df | sh
```
## Update
```shell
$ cd ~/.dotfiles
$ git pull
$ yes | ./install.sh
```
## Dependencies
- Common
    - Needed
        - `git`
- `.config/nvim`
    - Needed
        - `nvim`
        - `python-nvim`
    - Optional
        - `python-jedi`
        - `rust`
        - `rust-racer`
